## Skills y stellar-dev

Las **skills** son capacidades reutilizables para agentes. **stellar-dev** es la skill oficial del ecosistema Stellar.

- **Directorio abierto:** [skills.sh](https://skills.sh/) — catálogo de skills para agentes de IA.
- **stellar-dev:** [skills.sh/stellar/stellar-dev-skill/stellar-dev](https://skills.sh/stellar/stellar-dev-skill/stellar-dev)

**Instalación:**

```bash
npx skills add stellar-dev
```

> En el deck original también se mencionan deeplinks de Cursor para instalar otras skills relacionadas con la identidad visual de Stellar; ese contenido es de diseño/ilustración y no se reproduce en estos apuntes.
